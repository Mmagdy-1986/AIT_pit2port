# AIT Rental Equipment OS - Professional PWA

نسخة احترافية لتطبيق إدارة شركات التأجير والمقاولين والمعدات والعقود والمستحقات والورديات.

## ما تم تطويره
- شاشة Dashboard احترافية بمؤشرات مالية وتشغيلية وتنبيهات.
- إضافة شركة / مقاول من نموذج كامل بدل زر ديمو.
- حفظ البيانات الأساسية، القانونية، البنكية، جهات الاتصال، المرفقات وملاحظات المستندات.
- إضافة معدة مع بيانات التشغيل، المشغلين، المساعدين، الاستيراد، الوقود، الصيانة والمرفقات.
- إنشاء عقد إلكتروني مستقل مع اختيار الشركة والمعدة.
- دعم التسعير اليومي، بالساعة، وبالمتر المكعب.
- احتساب سعر الطن تلقائياً من سعر المتر المكعب وكثافة الخام.
- دورة سداد يومية / أسبوعية / شهرية مع موعد استحقاق تلقائي.
- شاشة مالية لتسجيل الدفعات وتحديث الرصيد المتبقي.
- عقد PDF مطبوع من المتصفح يشمل بيانات الشركة، المعدة، التسعير، السداد، الالتزامات والتوقيعات.
- تقارير تشغيلية ومالية وتقارير عقود.
- Dark Mode و PWA قابل للنشر على GitHub Pages.

## التشغيل محلياً
افتح `index.html` مباشرة من المتصفح أو شغل أي static server داخل المجلد.

## ملاحظات مهمة
هذه نسخة Frontend/PWA تعمل بـ localStorage. للإنتاج الحقيقي يفضل ربطها بـ Backend وقاعدة بيانات وصلاحيات مستخدمين ورفع ملفات فعلي.

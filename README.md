# AIT Rental Management PWA

واجهة Web App / PWA قابلة للنشر على GitHub Pages لإدارة شركات التأجير والمقاولين والمعدات والعقود والورديات والمستحقات.

## طريقة الرفع على GitHub Pages
1. أنشئ Repository جديد على GitHub.
2. ارفع كل الملفات الموجودة في هذا المجلد إلى الـ repository.
3. افتح Settings ثم Pages.
4. اختر Deploy from branch.
5. اختر branch: main و folder: /root.
6. افتح رابط GitHub Pages بعد دقائق.

## ملاحظات
- هذه نسخة Front-End تعمل من غير Backend.
- البيانات التجريبية تحفظ في LocalStorage داخل المتصفح.
- لنسخة إنتاج حقيقية يجب ربط Firebase أو Supabase أو Google Sheets Apps Script.
- الطباعة إلى PDF متاحة من شاشة التقارير عبر زر طباعة / حفظ PDF.

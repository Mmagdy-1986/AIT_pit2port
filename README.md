# AIT Rental Mobile PWA

نسخة محدثة من تطبيق إدارة شركات التأجير والمقاولين والمعدات والعقود.

## أهم التعديلات
- تصميم موبايل فعلي شبيه بالتطبيقات، وليس شكل تقرير PDF.
- Dashboard بكروت وأيقونات وBottom Navigation وFAB.
- Dark Mode.
- زر لغة AR/EN.
- بيانات ديمو لشركات ومعدات وعقود.
- صفحة تفاصيل الشركة تعرض عدد المعدات، المعدات الشغالة، العقود، والبيانات البنكية والمستندات.
- صفحة تفاصيل المعدة تعرض الحالة، الموقع، العقد، وعدد الأيام المتبقية.
- عند حفظ معدة جديدة يتم إنشاء عقد ديمو تلقائي وفتح معاينة العقد قبل الطباعة.
- معاينة عقد من 5 صفحات بنفس هوية عقد AIT PDF: هيدر، شعار، سفينة، watermark، footer، جدول تكاليف، توقيعات.
- زر طباعة / حفظ PDF من المتصفح.

## النشر على GitHub Pages
1. ارفع محتويات هذا المجلد في Repository.
2. افتح Settings > Pages.
3. اختر Deploy from branch.
4. اختر main / root.
5. افتح رابط GitHub Pages.
